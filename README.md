# Lycos

Utilitaire en ligne de commande pour trouver des fichiers

## Installation
```
$ wget https://raw.githubusercontent.com/npotier/lycos/master/lycos.sh -O ./lycos.sh && chmod +x ./lycos.sh && mv ./lycos.sh /usr/local/bin/lycos
```
